# Machine Learning Foundations 🧠⚙️

This repository contains my personal journey implementing core Machine Learning algorithms entirely from scratch. 

Instead of relying on high-level frameworks like Scikit-Learn or TensorFlow right away, the goal of these projects is to understand the underlying mathematics (Calculus and Linear Algebra) that power Artificial Intelligence.

## 🚀 Learning Objectives
* Understand and implement **Cost Functions** (Mean Squared Error).
* Build optimization algorithms like **Gradient Descent** from the ground up.
* Handle data manipulation and vectorization using **NumPy**.

## 📂 Repository Structure

* `01_Linear_Regression_Scratch/`
  * Contains a complete implementation of Univariate Linear Regression.
  * The model calculates the error, computes derivatives (Gradient), and updates weights ($w$, $b$) autonomously to minimize cost.

## 🛠️ Technologies & Tools
* **Python 3**
* **NumPy** (for mathematical operations)
* **Jupyter Notebook** (for interactive development and documentation)
